## Title of the Experiment
Design of a Product Filtering Interface using HTML and CSS

---

## Defined Approach
- Structured the product list using basic HTML elements.
- Used radio buttons to allow category selection.
- Applied CSS to hide and show products based on selected radio buttons.
- Avoided the use of JavaScript to implement filtering logic.
- Ensured the layout remains simple and easy to understand.

---

## Learning Outcomes
- Learned how to use form elements like radio buttons.
- Understood CSS-based conditional display techniques.
- Improved knowledge of CSS selectors and visibility control.
- Gained confidence in building logic without JavaScript.
